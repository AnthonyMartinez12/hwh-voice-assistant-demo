# HWH Voice Assistant — Evaluation Site

A single-page site for evaluating a proof-of-principle AI assistant built for **Healed Women Heal**, a Claremont-based nonprofit serving women, families, and individuals with developmental disabilities impacted by domestic violence.

## What this is

A working prototype, built in AI Squared's UNIFI platform during a Claremont Graduate University (CGU) volunteer engagement. The assistant uses Anthropic's Claude Haiku model, grounded in five HWH-flavored sample documents covering the organization's mission, voice, donor segments, a sample appeal, and responsible AI principles.

The site embeds the live chatbot and provides a structured set of evaluation prompts so reviewers can judge whether the approach is worth carrying into a production build.

## What this is not

This is not the production version. It uses sample content, not HWH's actual past appeals or donor records. Everything generated is a draft for human review — nothing is ever sent automatically.

## Hosting

This site is hosted via GitHub Pages from this repository. To view it, open the published Pages URL (Settings → Pages once deployed).
